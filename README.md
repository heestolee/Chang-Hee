  > ##### 개인 프로젝트 (총 3주, 1주 기획 / 2주 핵심 기능 개발)
  >
  > 스크랩북처럼 여러 페이지에서 취사선택한 정보를 하나의 정적 웹 페이지로 배포하는 웹 어플리케이션입니다.<br>
